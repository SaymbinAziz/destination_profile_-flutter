# Destination Profile - Flutter App

## Overview
Destination Profile is a Flutter app that shows information about different travel destinations. It focuses on a clean and simple user interface with smooth animations and easy navigation. The app is built to be responsive on both Android and iOS devices.

## Features
- View detailed information about each destination
- Hero image animations and interactive UI components
- Custom themes and color schemes
- Easy to extend with more destinations
- Works on different screen sizes
